# OpenAI GPT-Image-1

Мы подключили модель GPT-Image-1 для создания картинок. Она прославилась кадрами в стиле студии Ghibli.
Используйте короткие промпты с описанием настроения и композиции. Модель поддерживает редактирование через inpaint, поэтому можно доработать любые области.
