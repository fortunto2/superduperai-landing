## Описание
Расширение базовой компонентной библиотеки лендинга, добавление ключевых маркетинговых блоков и UI компонентов для создания полноценного лендинга.

## Технические требования
- Компоненты должны быть мобильно-адаптивными
- Использовать Tailwind CSS для стилизации
- Следовать принципам shadcn/ui для консистентности
- Каждый компонент должен быть типизирован через TypeScript
- Компоненты должны иметь понятные пропсы и документацию

## План реализации
1. **Этап 1: Маркетинговые секции**
   - Создать компонент Features (список преимуществ)
   - Создать компонент Testimonials (отзывы пользователей)
   - Создать компонент CTA (призыв к действию)
   - Создать компонент Pricing (тарифы)
   
2. **Этап 2: UI элементы**
   - Добавить кнопки с различными вариантами
   - Добавить компонент Card с вариантами
   - Добавить компонент Badge для меток
   - Добавить базовые элементы формы
   
3. **Этап 3: Композиция лендинга**
   - Интегрировать все компоненты на главную страницу
   - Создать связанную композицию между блоками
   - Убедиться в консистентности стилей

## Критерии готовности
- [ ] Все запланированные компоненты созданы и работают
- [ ] Компоненты адаптивны для мобильных экранов
- [ ] Главная страница отображает полноценный лендинг
- [ ] Стили компонентов консистентны и следуют дизайн-системе
- [ ] Все компоненты имеют документацию в виде комментариев
- [ ] Код проходит линтинг и TypeScript проверки
