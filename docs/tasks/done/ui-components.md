## Описание
Необходимо внедрить библиотеку shadcn/ui в проект для единого стиля интерфейса, настроить поддержку темной темы и добавить базовые компоненты для повторного использования во всем проекте.

## Технические требования

### Основные шаги
1. **Инициализация shadcn/ui**
   - Интеграция с Tailwind CSS 4.x
   - Выбор темы "new-york" (рекомендуемая)
   - Настройка CSS переменных для цветовой схемы

2. **Настройка темной темы**
   - Реализация ThemeProvider
   - Создание компонента переключения темы
   - Интеграция в Layout

3. **Компоненты для добавления**
   - Button
   - Card
   - Dropdown Menu
   - Sheet
   - Form компоненты
   - Dialog
   - Toggle
   - Theme Toggle

### Расположение файлов
```
src/components/ui/button.tsx
src/components/ui/card.tsx
src/components/ui/dropdown-menu.tsx
src/components/ui/sheet.tsx
src/components/theme-provider.tsx
src/components/theme-toggle.tsx
```

### Общие требования
- Использовать последнюю версию shadcn/ui, совместимую с Tailwind CSS 4.x и React 19
- Все компоненты должны поддерживать темную тему
- Убедиться в корректной настройке атрибута темы (`data-theme="dark"` или `class="dark"`)
- Адаптировать код для React 19 (убрать forwardRefs, где это необходимо)

## План реализации
1. **Этап 1: Инициализация shadcn/ui**
   - Установить CLI: `npx shadcn-ui@latest init`
   - Выбрать опции: TypeScript, тема "New York", цвет "Slate"
   - Указать путь до глобального CSS: `src/app/globals.css`
   - Настроить поддержку темной темы
   
2. **Этап 2: Компоненты темной темы**
   - Создать ThemeProvider
   - Создать компонент переключения темы
   - Интегрировать в макет приложения
   
3. **Этап 3: Добавление базовых компонентов**
   - Добавить кнопку: `npx shadcn-ui@latest add button`
   - Добавить карточку: `npx shadcn-ui@latest add card`
   - Добавить выпадающее меню: `npx shadcn-ui@latest add dropdown-menu`
   - Добавить другие компоненты по необходимости

4. **Этап 4: Обновление существующих компонентов**
   - Адаптировать Header для использования новых компонентов
   - Добавить переключатель темы в Header
