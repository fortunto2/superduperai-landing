## Описание
Создание минимальной работающей версии лендинга на Next.js с использованием компонентов shadcn/ui. Этот этап служит фундаментом для дальнейшей разработки.

## Технические требования
- Использовать Next.js 15.x (App Router)
- Настроить Tailwind CSS 4.x
- Интегрировать shadcn/ui
- Настроить линтер ESLint
- Обеспечить успешную сборку проекта

## План реализации
1. **Этап 1: Базовая структура проекта**
   - Настроить основные директории проекта согласно frontend-tech.md
   - Создать базовый layout с корректным HTML и метаданными
   - Создать простую главную страницу
   
2. **Этап 2: Настройка Tailwind CSS и shadcn/ui**
   - Настроить Tailwind CSS 4.x следуя frontend-tech.md
   - Интегрировать необходимые компоненты shadcn/ui
   - Создать простой Hero компонент
   
3. **Этап 3: Линтинг и проверка сборки**
   - Настроить ESLint и Prettier
   - Проверить успешную сборку проекта (pnpm build)
   - Исправить возможные ошибки

## Критерии готовности
- [ ] Проект успешно запускается локально (pnpm dev)
- [ ] Проект успешно собирается (pnpm build)
- [ ] На главной странице корректно отображается Hero компонент
- [ ] Линтер не выдает ошибок (pnpm lint)
- [ ] TypeScript проверка проходит без ошибок (pnpm type-check)
