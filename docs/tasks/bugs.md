# Известные ошибки и их решения

## Tailwind CSS

### Ошибка: `Cannot apply unknown utility class: border-border` или `bg-background`

**Проблема**: При запуске приложения возникает ошибка Tailwind CSS о невозможности применить неизвестный utility класс, например `border-border` или `bg-background`.

**Решение**:
1. Заменить `@apply border-border` на `border-color: hsl(var(--border))`
2. Заменить `@apply bg-background` на `background-color: hsl(var(--background))`
3. Вообще избегать использования `@apply` с нестандартными классами, особенно теми, которые используют CSS переменные

**Техническое объяснение**:
Директива `@apply` работает только со стандартными классами Tailwind или с классами, которые четко определены в `@layer`. Когда вы используете `@apply` с именем, которое не является стандартным классом (например, `border-border`), Tailwind не может найти его в своем реестре классов.

В случае с переменными цвета в темах, правильный подход - использовать нативные CSS свойства с функцией `hsl()`:

```css
/* НЕ ПРАВИЛЬНО */
* {
  @apply border-border;
}

/* ПРАВИЛЬНО */
* {
  border-color: hsl(var(--border));
}
```

Также убедитесь, что в `tailwind.config.js` или `tailwind.config.mjs` используется формат `hsl(var(--color-name))`:

```js
// ПРАВИЛЬНО
colors: {
  border: "hsl(var(--border))",
  background: "hsl(var(--background))",
  // остальные цвета...
}

// НЕ ПРАВИЛЬНО
colors: {
  border: "var(--border)",
  background: "var(--background)",
  // остальные цвета...
}
```
